Steps to commit a file
----------------------
1. CREATE A FILE		------> touch filename
2. NOW ADD THAT FILE		------> git add filename
3. COMMIT THE FILE WITH MESSAGE	------> git commit -m "TYPE YOUR MESSAGE" filename 
4.TO SEE THE DETAILS OF THE FILE------> git log
